=== About Laurels ===
Laurels is a lightweight, faster, clean, super flexible and bootstrap based responsive theme for business or personal websites. Laurels Theme is built on top of Twitter Bootstrap which means that your site can be displayed on every device (pc, laptop, mobile devices, tablets etc) without horizontal scrolling. It has got custom widgets, theme-options etc in order to get it customized as per your needs.

== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "laurels.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (laurels.zip) that you have downloaded.
 = Upload the entire folder (laurels) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/laurels/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "laurels" to activate the theme.

5. one custom widgets :
	i) laurels recent post widget for display recent post in sidebar or footer.
	
6. Theme Options :

	a) You can manage here site logo,favicon,social icon and footer copyright text.	

	b) Home Page Settings
		i) Banner Slider : Here you can manage home page slider. 
		ii) Title Bar : You can set home page title and description.
		iii) First Section : you can upload image, title, post and shortdescription. 
		iv) Second Section : you can set category post title and category.
		v) Third Section : If you want to change latest post title then you can change it.
